# cns-pc-guide

## 项目简介

cns-pc-guide 是一个用于归档与发布多个独立 HTML 页面的仓库。这些页面通常涉及使用说明、配置参考、本地化指引或操作指南等通用内容，不针对任何特定网站或域名。

本仓库旨在提供一个结构清晰、易于访问的静态页面集合，方便用户通过 GitHub Pages 或其他静态托管服务查看。

## 目录说明

```
cns-pc-guide/
├── docs/          # 存放所有独立 HTML 页面及相关资源
├── assets/        # 静态资源文件（如 CSS、JS、图片等）
└── README.md      # 本文件
```

- `docs/`：主要归档目录，每个 HTML 文件为一个独立页面。
- `assets/`：页面所需的公共资源文件，按需组织。

## 页面归档说明

- 每个 HTML 页面均为自包含的独立文档，可单独访问。
- 页面内容以通用性为主，不涉及特定商业或敏感信息。
- 页面命名尽量简洁，反映内容主题。
- 部分页面可能引用 `assets/` 下的资源，请确保路径正确。

## 维护说明

- 欢迎提交 Issue 或 Pull Request 补充或修正页面内容。
- 提交新页面时，请放入 `docs/` 目录，并保持命名风格一致。
- 资源文件请放入 `assets/` 对应子目录，避免与页面文件混杂。
- 本仓库不接收任何包含广告、推广或特定域名链接的内容。

## 使用方式

1. 克隆本仓库到本地。
2. 直接打开 `docs/` 下的 HTML 文件，或通过本地服务器预览。
3. 若启用 GitHub Pages，访问 `https://<用户名>.github.io/cns-pc-guide/` 可查看首页。
